# Privacy Statement

This privacy statement applies to the Power BI custom visual “Advanced Pie Chart”.

## Summary

- No external data transmission: the visual does not send report data to any external service.
- No telemetry/analytics: the visual does not collect usage telemetry.
- No persistent storage of report data: the visual does not store report data in browser localStorage/sessionStorage.

## What data the visual processes

The visual processes only the data that Power BI provides to it for rendering (for example, category names and measure values selected in the visual fields).

## What data the visual collects

The visual does not intentionally collect:

- Personal information (PII)
- Device identifiers
- Location information
- Usage analytics or tracking events

## Data sharing

The visual does not share data with third parties.

## Data retention

The visual does not persist report data. Data is processed in-memory for rendering and is discarded when Power BI refreshes or the visual is removed.

## Changes to this policy

If the visual’s behavior changes in a way that affects privacy, this statement should be updated accordingly.
